* Angular.js - scope, filters, directives, routes, controllers, services, factory, config
* Express.js
* Jasmine framework
* karma cli
* Protractor end to end testing framework
* PhantomJS
* ngMock
* Bootstrap
* HTML5
* Gulp
* Grunt (config also available)
* Bower
* Node.js
* json-server
